# Tournament_service

**Цель:** создать веб-сервис для организации и проведения турнира.
# Функциональные требования участника турнира
1) Авторизация в системе
2) Выбор турнира
3) Регистрация на турнир
4) Отмена регистрации
5) Просмотр сетки
6) Просмотр профиля
7) Просмотр своих раундов
8) Просмотр любого турнира

# Функциональные требования админа
1) Авторизация в системе
2) Выбор турнира
3) Создание турнира
4) Редактирование турнира
5) Просмотр участников, зарегистрировавшихся на турнир
6) Подтверждение заявок
7) Исключение участника
8) Генерация турнирной сетки
9) Управление матчами (изменение результатов)
10) Просмотр турнирной сетки
11) Просмотр любого турнира
12) Просмотр профиля любого участника
13) Поиск и фильтрация


# MVP
## Участник
1) Авторизация
2) Просмотр турнира
3) Регистрация на турнир
4) Просмотр сетки
5) Просмотр профиля

## Админ
1) Авторизация
2) Просмотр турнира
3) Создание и редактирование турнира
4) Просмотр участников
5) Модерация участников
6) Генерация турнирной сетки
7) Управление матчами
8) Просмотр профиля
